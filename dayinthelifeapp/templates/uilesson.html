Good morning class! Today we will be learning the basics of HTML.

HTML stands for Hyper Text Markup Language. This is just a fancy way

to describe links. This concept is at the core of the world wide web and

without HTML you can not have a web page. Think of it like the skeleton

of a web page. This skeleton is made up of elements, or tags.

Within these tags is where the content of the page is stored.

Here is an example of an opening and a closing tag: <h1></h1>

Ready to try it yourself?
